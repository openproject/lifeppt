# 小学生手卫生 GPT Image 2 套图

本目录是一套面向小学二年级的 16:9 手卫生科普图片。

- `final/`：叠加准确中文文案后的最终 1920x1080 PNG，可直接用于课件或导入 PPT。
- `source/`：GPT Image 2 生成的无文字插图底图，便于二次调整；如果需要带文字版本，请使用 `final/`。
- `compose_gpt_image2_slides.py`：合成脚本，可调整每页文字、字号与位置后重新生成。

## 页面结构

1. 封面页：小手洗一洗 健康伴我行；副标题：-- 冯可为妈妈
2. 目录页：四大学习模块清晰展示
3. 第一章：为什么要洗手 - 认识手上的细菌
4. 第一章：为什么要洗手 - 不洗手的后果
5. 第二章：什么时候要洗手 - 五大关键时刻
6. 第三章：怎样正确洗手 - 七步洗手法总览
7. 第三章：怎样正确洗手 - 七步洗手法图解 1-4
8. 第三章：怎样正确洗手 - 七步洗手法图解 5-7
9. 第三章：怎样正确洗手 - 洗手小贴士
10. 结尾页：洗手小达人宣誓

## 调整方式

如需调整文字或布局，编辑 `compose_gpt_image2_slides.py` 中 `SLIDES` 配置，再运行：

```bash
python3 手卫生/gpt_image2_slides/compose_gpt_image2_slides.py
```

最终图片会重新输出到 `final/`。
